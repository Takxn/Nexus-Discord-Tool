üîß Discord Developer Portal Setup

These settings must be enabled in the Discord Developer Portal.

Privileged Gateway Intents

Go to your Application ‚Üí Bot ‚Üí Privileged Gateway Intents

‚úÖ REQUIRED
PRESENCE INTENT Allows the bot to see online status of members
‚úÖ REQUIRED
SERVER MEMBERS INTENT Allows the bot to see member list and receive join/leave events
‚úÖ REQUIRED
MESSAGE CONTENT INTENT Allows the bot to read message content (for AutoMod, Leveling, etc.)
Without these intents you will get the error: Used disallowed intents

üõ°Ô∏è Bot Permissions

These permissions are required when inviting the bot.

General
  • View Channels - See channels
  • Manage Channels - Manage channels (for tickets)
  • Manage Roles - Manage roles (for reaction roles)
  • View Audit Log - View audit log
  • Manage Server - Manage server (for invites)
Members
  • Kick Members - Kick members
  • Ban Members - Ban members
  • Moderate Members - Timeout members
  • Manage Nicknames - Change nicknames
Messages
  • Send Messages - Send messages
  • Send Messages in Threads - Send in threads
  • Embed Links - Send embeds
  • Attach Files - Attach files
  • Add Reactions - Add reactions
  • Manage Messages - Delete messages
  • Read Message History - Read message history
  • Mention Everyone - Mention @everyone
Voice
  • Connect - Connect to voice
  • Speak - Speak in voice
  • Move Members - Move members
  • Mute Members - Mute members
Tip: Use permission integer 8 for Administrator rights (all permissions) when inviting.
